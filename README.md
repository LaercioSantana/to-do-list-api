# To-do list

**Environment**
  
 * Ruby 2.4.0
 * Rails 5.0.2

### Usage

On the root folder:

```console

$ bundle install     # install dependencies
$ rake db:create     # create database
$ rake db:migrate    # update database schema
$ rails s            # run server

```

## Run tests

On the root folder:

```console
$ rspec spec --format documentation
```
